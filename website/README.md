# Steffi Metz - Static Website

A modern, responsive static website for Steffi Metz restaurant in Rwanda, featuring handmade healthy cuisines from around the world with Rwandan touches.

## Features

- Fully responsive design (mobile, tablet, desktop)
- Beautiful Rwandan-inspired color scheme (green, yellow, blue, red)
- Smooth animations and transitions
- Clean, modern UI with Tailwind CSS
- Sections: Home, About, Menu, Gallery, Contact

## Quick Start (Development)

1. Open `index.html` directly in your browser - it uses Tailwind CSS CDN for instant preview

## Production Build

For optimized production deployment:

```bash
# Install dependencies
npm install

# Build optimized CSS
npm run build
```

This generates `dist/styles.css` which can replace the CDN version in production.

## Deployment

Upload all files to your hosting via cPanel File Manager or FTP:
- `index.html`
- Any images (or use external CDN)
- `dist/styles.css` (if using compiled version)

## Tech Stack

- HTML5
- Tailwind CSS (CDN for dev, compiled for production)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Poppins)
